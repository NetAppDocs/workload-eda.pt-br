= Ative o painel de Visão Geral no Workload Factory para EDA
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Ative o painel Visão geral para começar a coletar métricas do CloudWatch para seus sistemas de arquivos FSx for ONTAP. O painel permite que você veja a integridade do cluster, a capacidade e a taxa de transferência em um só lugar.



[[before-you-begin]]
== Antes de começar

Certifique-se de atender aos seguintes requisitos antes de ativar o dashboard.



[[aws-credentials-with-view-permissions]]
=== Credenciais da AWS com permissões de visualização

Você deve configurar as credenciais da AWS no Workload Factory com pelo menos permissões de leitura (visualização) para General Storage. Credenciais _básicas_ não são suportadas.

Se você não configurou credenciais com permissões de visualização, você será redirecionado para a página de configuração de credenciais da AWS ao abrir a guia Overview.

Se você ainda não configurou as credenciais AWS, consulte link:https://docs.netapp.com/us-en/workload-setup-admin/add-credentials.html["Adicionar credenciais da AWS"^].



[[activate-the-dashboard]]
== Ative o dashboard

Após o Workload Factory verificar as credenciais com permissões de visualização, ative o painel para começar a coletar métricas do CloudWatch para seus sistemas de arquivos FSx for ONTAP.


NOTE: Após você dar seu consentimento, a coleta de métricas pode levar algum tempo. O painel de controle notificará você enquanto a primeira coleta estiver em andamento.

.Passos
. Faça login usando um dos seguintes métodos: link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["experiências de console"^].
. Selecione o menu image:workloads-menu-icon.png["O ícone do menu hambúrguer"] e depois selecione *EDA*.
. Selecione a guia *Visão geral*.
. Se nenhuma credencial com permissões de visualização for detectada, selecione *Add credentials* e siga as instruções para configurar as credenciais da AWS com permissões de visualização. Em seguida, retorne à guia *Overview*.
. Analise o aviso de consentimento que descreve as métricas CloudWatch que serão coletadas para seus sistemas de arquivos FSx for ONTAP.
. Selecione *Ativar* para ativar o painel e iniciar a coleta de métricas.


.Resultado
O Workload Factory começa a coletar métricas do CloudWatch para todos os sistemas de arquivos FSx for ONTAP vinculados às suas credenciais da AWS. O painel é preenchido conforme as métricas ficam disponíveis. Uma notificação aparece enquanto a coleta de dados ainda está em andamento.
